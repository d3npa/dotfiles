# dotfiles

- OS: [Void Linux](https://voidlinux.org/) (x86_64/glibc)
- Sudo: [opendoas](https://github.com/Duncaen/OpenDoas)
- WM: [dwm](https://dwm.suckless.org/)

- Brightness control: [light](https://haikarainen.github.io/light/)
